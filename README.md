### 1、分布式会话-登录功能总结

0、两次 md5 加密

1、一开始通过参数传递 mobile、password 需要对参数进行判断和校验

2、使用 @Valid 进行参**数校验**，以及自定义参数校验

3、校验失败时有异常，进行**全局异常处理**

4、**分布式 session** 问题，两种方式

5、每一个接口都需要做参数的校验，自定义 **用户参数** ，==在 controller 入口之前就做好了一层校验==
